# Objetivo Final

Entregar un Sistema de Titulación completamente funcional que permita administrar estudiantes, docentes, asesores, jurados, proyectos de investigación, expedientes y sustentaciones mediante una plataforma web desarrollada en Laravel y MariaDB.
